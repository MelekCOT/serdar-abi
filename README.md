# İkra Scooter Rent & Kurye

Scooter ve moto kiralama, motokurye ve arabalı kurye hizmetleri için tek sayfalık web sitesi.

## Özellikler

- 🛵 Scooter, moto kiralama
- ⚡ Motokurye hizmeti
- 🚗 Arabalı kurye hizmeti
- 🌐 Türkçe / İngilizce dil desteği
- 📱 WhatsApp iletişim butonu
- 📸 Galeri şeridi
- 📄 KVKK uyumlu gizlilik politikası

## Kurulum

1. Projeyi indirin veya klonlayın
2. `config.js` dosyasında telefon numaranızı güncelleyin
3. `images` klasörüne galeri görsellerini ekleyin (gallery1.png - gallery6.png)
4. `index.html` dosyasını tarayıcıda açın

## Teknolojiler

- HTML5, CSS3, JavaScript
- Responsive tasarım
